## Overview
Web application to manage customer Purchase Orders (PO) and internal Sales Orders (SO), focusing on tracking delivery quantity and mapping product codes. Includes a Smart PO Suggestion feature to prioritize older inventory.

## Tech Stack
- Frontend: Next.js (App Router), TailwindCSS, Shadcn UI
- Backend: Next.js API Routes (Serverless) / Node.js
- Database: PostgreSQL (Recommended for structured data) or SQLite (Local file)
- Auth: NextAuth.js (Simple manual login)
